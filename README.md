# Churn Prediction System

A full-stack **Machine Learning web application** that predicts whether a customer is likely to churn or not.  
This project combines **ML model training**, a **FastAPI backend**, and a **React frontend** to deliver predictions through a clean and interactive UI.

---

## 🚀 Features

- Predicts customer churn using trained ML models
- Built with a **FastAPI backend**
- Interactive **React + Tailwind CSS frontend**
- Uses **Logistic Regression** and **XGBoost** for training and prediction
- Trained on a **Kaggle Customer Churn dataset**
- Simple and user-friendly interface for entering customer details
- Real-time prediction response from backend API
